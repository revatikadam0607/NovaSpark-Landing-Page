## 📖 About

NovaSpark is a modern startup-inspired landing page built using pure HTML, CSS, and JavaScript. It focuses on elegant design, responsiveness, and interactive elements to create a premium user experience.

---

## ✨ Features

* 🎨 Modern and Minimal UI
* 📱 Fully Responsive Design
* 🌙 Dark Theme
* ⚡ Smooth Scroll & Animations
* 🧊 Glassmorphism Effects
* 🚀 Hero Section
* 💼 Services Section
* ⭐ Customer Testimonials
* 📬 Contact Section
* 🎯 Interactive Navigation
* 🔥 Clean and Organized Code

## 🛠️ Tech Stack

* HTML5

* JavaScript (Vanilla)
